This WebApp is a part of my M.Eng. project in the University of Toronto. This page briefly introduces the ASM and newly trianed ASM in my project that can be utilized in a wide branch of applications.

The active shape model (ASM) is considered as a widely used method for facial feature detection, and it defines a set of labeled facial landmarks that form a face shape. In the early version, it combines grey-scale information for extracting a one-dimensional gradient profile around each landmark to describe surrounding texture. The active shape model is considered as a widely used method for facial feature detection, and it defines a set of labeled facial landmarks that form a face shape. In the early version, it combines grey-scale information for extracting a one-dimensional gradient profile around each landmark to describe surrounding texture.

But this approach fails to have a satisfactory precision when it comes to complex facial texture structure like eye corner or boundary. S. Milborrow et al. They proposed an optimized ASM called Stasm that introduced a new template matching method and an adaptive descriptor matching method.However, the dataset used for training ASM is also a key factor impacting the performance, especially for practical applications. Besides, as the development of image processing and recognition, there is a incresing need for high-resolution images and the model with more landmarks.

The Helen dataset created for training reliable, accurate algorithm meets the requirement of both variation of appearance and high resolution. The Helen dataset consists 2000 training images and 330 test images. For comparing the performance between two models, the new model trained on the Helen dataset is evaluated on BioID dataset. Meanwhile, some changes and optimizations of the source code of Stasm are made for 194 landmarks shape and high-resolution images in Helen dataset. In a comparative experiment, these two ASMs are examined on performance in various lighting conditions and with different interference factors.
